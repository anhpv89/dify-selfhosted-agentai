# Dify Self-Hosted Infrastructure by AgentAI

This repository contains the optimized infrastructure deployment configuration for self-hosting **Dify** on a private Cloud VPS, prepared by [AgentAI](https://agentai.io.vn).

## Purpose
As part of our technical review and deployment guide on AgentAI, this configuration focuses on:
- High availability and stability for 24/7 AI pipelines.
- Security configurations for SysAdmins deploying Dify in production.
- Optimized resource allocation for Vector DB and PostgreSQL.

## How to deploy
*(Full sysadmin tutorial will be published on agentai.io.vn)*
1. Clone this repository to your VPS.
2. Configure your `.env` file based on the provided environment variables.
3. Run `docker compose up -d` to deploy the complete Dify stack.
